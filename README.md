# Cosmic Archive

A data-driven periodic table website featuring all 118 elements with interactive learning tools.

## Features

- JSON-based element data
- JavaScript template system
- Individual pages for each element
- Interactive periodic table with period colors and filtering
- Element search with highlight and scroll
- Element quiz system
- Interactive popup feature
- Element comparison tool
- Atom visualizer with Bohr models
